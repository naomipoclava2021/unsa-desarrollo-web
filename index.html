Bootstrap - Container

Los contenedores son el elemento de dise√±o mas basico en Bootstrap y son necesarios para utilizar el sistema de rejillas predeterminado.

Se utiliza para contener, rellenar y (a veces) centrar el contenido dentro de ellos.

Bootstrap viene con tres contenedores diferentes:
.container, proporciona un contenedor de ancho fijo.
.container-fluid, proporciona un contenedor de ancho completo, que abarca todo el ancho del viewport.
.container-{breakpoint}, abarca un ancho de 100% hasta el punto de quiebre (breakpoint)
