Selectores de CSS

Los selectores de CSS se utilizan para "encontrar" (o seleccionar) los elementos HTML que se quieren diseñar.

Selectores simples - seleccionar elementos basados en nombre, id, Clase
Selectores combinadores - seleccionar elementos en funcion de una relacion especifica entre ellos
Selectores de pseudoelementos - seleccionar y diseñar una parte de un elemento
Selectores de atributos - seleccionar elementos en funcion de un atributo o valor de atributo.

Selectores de Pseudoelementos

Un pseudoelemento se utiliza para aplicar estilo a partes especificas de un elemento. Por ejemplo, se puede utilizar para:
- Aplicar estilo a la primera letra, o linea, de un elemento.
- Insertar contenido antes o despues del contenido de un elemento.

::first-latter*-- para agregar un estilo especial a la primero letra de un texto.
::first-line*-- para agregar un estilo especial a la primera linea de un texto.
::before-- para insertar algun contenido antes del contenido de un elemento.
::after-- para insertar algun contenido despues del contenido de un elemento.

*solo se puede aplicar a elementos a nivel de bloque.

Selectores de Atributos

Permite aplicar estilo a elementos HTML que tengan atributos o valores de atributo especificados.
[attribute] --selecciona elementos con un atributo especifico.
[attribute=value] --selecciona elementos con un valor de atributo que contiene una palabra especifica.
[attribute~=value] --selecciona elementos con un valor de atributo que contiene una palabra especifica.
[attribute|=value --selecciona elementos con el atributo especificado, cuyo valor es exactamente el valor especificado, o el valor especificado seguido de un guion (-).
[attribute^=value] --selecciona elementos con el atributo especificado, cuyo valor comienza con el valor especificado.
attribute$=value] --selecciona elementos cuyo valor de atributo termina con un valor especificado.
[attribute*=value] --selecciona elementos cuyo valor de atributo contine un valor especificado.

Viewport

El viewport es el area visible que tiene el usuario de una pagina web, y cambia segun el dispositivo.

HTML5 mediante la etiqueta meta, permite que el navegador establezca el ancho del viewport igual al ancho de la pantalla del dispositivo actual. El valor de escala inicial establece el nivel de zoom en 1 (100%).
